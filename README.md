## Installation ##

### Cocoa Pods ###

*Good Resource: [Use CocoaPods with Swift](https://www.raywenderlich.com/97014/use-cocoapods-with-swift)*

```
sudo gem install cocoapods
```

```
pod setup --verbose
```

```
pod install
```

Dependencies:

* [LockSmith](https://github.com/matthewpalmer/Locksmith)
